# MTBS_movieticketbookingsystem
Movie Ticket Booking System (Python)  A simple and interactive Python console-based application that allows users to view available movies, book tickets, and cancel booked tickets. This project is perfect for beginners learning Python, loops, dictionaries, and functions.

 Features
 View Available Movies
 Displays all movies with:
 Movie Name
 Ticket Price
 Available Seats
 Book Movie Tickets
 Select a movie by ID
 Enter number of seats to book
 Validates seat availability

Updates remaining seats
Shows total bill
Cancel Tickets
Select a movie by ID
Enter seats to cancel
Seats get added back safely

User-Friendly Menu
Simple dashboard with:
Show Movies
Book Ticket
Cancel Ticket
Exit

 Tech Used
 Python (Core language)
 Dictionaries for data storage
 Functions & Loops for operations
 Input Handling
